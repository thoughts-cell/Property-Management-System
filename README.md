# Property-Management-System
A 3-tier enterprise application to manage properties which integrates presentationtier, business tier, and persistence tier.
In the persistence tier, the property and property manager information were storedin database using Jakarta Persistence API (JPA).The business tier processes the datapersistence or retrieval requests from users and interact with the persistence tier for
accomplishing the requests.
Tools: Java ,Jakarta EE, JPA ,JSF ,EJB ,MySQL,Maven
